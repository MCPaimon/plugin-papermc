# MCPaimon — PaperMC

The PaperMC platform implementation of the MCPaimon Minecraft AI framework. Hosts
the AI provider and manages MCExtension plugins on PaperMC servers.

Builds the runnable `MCAgentsPaperMC` plugin jar (Shadow) and publishes
`io.github.mcpaimon:papermc` to GitHub Packages.

- Depends on `io.github.mcpaimon:api` and `io.github.mcpaimon:common` with static
  versions.

See [`AGENTS.md`](AGENTS.md) and the [`.agents/`](.agents) folder for contributor
instructions.
